## Использованные технологии

-   React

## Поддерживаемые браузеры

-   Chrome
-   Edge
-   Firefox ESR+
-   Internet Explorer 10+
-   Safari 8+
-   Opera

## Инструкция по установке:

```
git clone https://github.com/hsh01/react-mesto-auth.git
cd react-mesto-auth
cp .env.example .env
npm i --save-dev
npm start
```
